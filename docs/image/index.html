Images

Images can be responsive to fit the parent's width, and also can be customised to be round shaped.

Round & Curved Border Images

Images can be made round by using img-round class inside <img> tag. For curved border mages, add img-border-curve class. (e.g. class="img-border-curve")

Don't forget to add Responsive Image class names for img element.
